# Employee Management System (C++)

A menu-driven Employee Management System developed in C++ as a university project.

This application simulates employee operations inside a small startup company using file handling and structured programming concepts. The system reads employee data from a file and allows different management operations through an interactive console menu.

## Features

- Create company employee list from file
- Print all employee records
- Hire new staff with input validation
- Fire employees using Employee ID or Phone Number
- Prevent removal of executive employees
- Update employee information
- Sort employees by:
  - First Name
  - Last Name
  - Employee ID
  - Phone Number
  - Birth Year
- Search employees by salary range
- Display employees by gender
- Display departments with minimum or maximum employees
- Employee bonus eligibility checking
- Dynamic employee record management
